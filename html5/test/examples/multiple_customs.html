HTML code

Multiple players on one page. Different videos and different options. The first player is set to autostart muted. The second player has pre-set duration, but no flashplayer fallback. Both have a custom skin.

First, place this code in the <head> of your page:

<script type="text/javascript" src="/scripts/jquery.js"></script>
<script type="text/javascript" src="/scripts/jquery.jwplayer.js"></script>

Second, place this code where you want the first video to appear:

<video 
  height="270" 
  id="player1"
  poster="/files/bunny.jpg"
  src="/files/bunny.mp4"
  width="480" >
  <a href="/files/bunny.mp4">Click here</a> to download this video.
</video>

<script type="text/javascript">
  $('#player1').jwplayer({
    autostart:true,
    flashplayer:'/files/player.swf',
    mute:true,
    repeat:true,
    skin:'/files/icecreamsneaka.xml'
});
</script>

Third, place this code where you want the second video to appear:

<video
  height="270"
  id="player2"
  src="/files/corrie.mp4"
  width="480" >
  <a href="/files/corrie.mp4">Click here</a> to download this video.
</video>

<script type="text/javascript">
  $('#player2').jwplayer({
    duration:45,
    skin:'/files/stormtrooper.xml'
});
</script>

Note that the jwplayer() function is called twice. This allows us to set different options for the two players.
