Quick Start

Getting JW Player up and running is now easier than ever. Only a few lines of HTML code are needed to embed a video that works on all popular desktop browsers and mobile devices. This guide walks through the necessary steps.

1: Upload JW Player

Before embedding a video, you must upload JW Player to your website. Unzip your player download and upload the jwplayer folder to the wwwroot of your website. The following three files should now be accessible over the internet:

  • http://yoursite.com/jwplayer/jwplayer.js (the light-weight embed library)
  • http://yoursite.com/jwplayer/jwplayer.html5.js (the actual HTML5 player)
  • http://yoursite.com/jwplayer/jwplayer.flash.swf (the actual Flash player)

The download of the Free edition is found on the JW6 product site. Downloads of the Pro, Premium and Ads editions are found in your account dashboard.

2: Include JW Player

With the JW Player uploaded to your site, it now must be added to your code. Do this by linking jwplayer.js in the <head> of your HTML page:

<script type="text/javascript" src="/jwplayer/jwplayer.js" ></script>

Since the embed library is light-weight, it's no problem to simply include JW Player with every page on your site.

Note you don't have to include the library in the <head> of your page, as long as it is included before an actual video is embedded.

JW Player Key

If you have a JW Player account key, set it in a second line of code right below the first one:

<script type="text/javascript">jwplayer.key="YOUR_JW_PLAYER_KEY";</script>

The key enables tracking of video analytics to your account. It also unlocks the additional features of the Pro, Premium and Ads editions.

3: Embed a Video

With JW Player uploaded and included, it's now time to actually embed a video. Therefore, insert this HTML code at the location you want the video to appear:

<div id="myElement">Loading the player...</div>

<script type="text/javascript">
    jwplayer("myElement").setup({
        file: "/uploads/example.mp4",
        image: "/uploads/example.jpg",
    });
</script>

In this code, the <div> element is used as a container to load JW Player into. It must have a unique id attribute.

The setup() block defines which video file and poster image to load. Their URLs should be relative to the HTML page (just like embedding an images <img>).

Next Steps

That's it! With a few lines of code, you embedded a video that can be watched on any desktop browser, tablet or mobile phone. Next, JW Player supports many options for enhancing and tuning your embeds. Here are a few interesting guides:

  • Supported Media Formats lists which audio & video types are played on which browsers & devices.
  • Embedding JW Player lists the options for modifying the player's layout and behavior.
  • Working with Playlists explains how to group multiple videos (or qualities) into a playlist.
  • Adding Video Captions explains how to add closed captions or subtitles to your embeds.
  • Using JW Player Skins explains how to customize the player's design through skinning.
  • Using the JavaScript API explains how to build rich player interactions with JavaScript.
