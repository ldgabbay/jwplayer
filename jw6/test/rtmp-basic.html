RTMP Basic

List of test devices

This test includes playback of single files using RTMP. It also includes testing the bufferlength option

Various Servers

ActionExpected result(s)Works?
Start each item Each item should start playing.
The duration of each item should be displayed in the controlbar
For AAC and MP3, the poster should remain visible.
Pause each item and resume Each stream should resume from the current position.
Re-buffering should not occur upon resume.
Seek each item halfway The stream should resume from the new position.
Re-buffering should be swift and visible only once.

Buffer Length

ActionExpected result(s)Works?
Start the stream Bufferlength is 10 seconds, so buffering should take long.
