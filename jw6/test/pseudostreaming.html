Pseudostreaming

List of test devices

This page tests the Flash MP4 and FLV pseudostreaming capabilities.

Single FLV

ActionExpected result(s)Works?
Click to play Video should start playing and buffering.
FLV URL should be loaded without start parameter.
Seek forward inside buffer Video should jump to new position without re-buffer.
Seek forward outside buffer Video should jump to new position, buffer should restart.
Seek back to beginning Video should jump to new position, buffer should restart.
Quickly do several seeks,
then wait until complete		 Buffer and time should not exceed 100%, playback should not stall.
Replay video Video should play and buffer from start again.
Reload page, then seek to 05:00 Video should buffer, then start playing from 05:00.
