Playing a YouTube video

List of test devices

This test consists of a basic setup with a single YouTube video. A small but broad set of cases is tested to check basic YouTube playback functionalities.

ActionExpected result(s)Works?
Initial page load 480x200 poster image should be visible, with play icon in the middle
Initial page load YouTube player instead of JW Player should be visible on Android/iOS.
Fallback devices should display media not supported error.
Click/touch display to play Video should first buffer, then play.
YouTube logo should popup in the bottom right corner.
HD icon should pop up and get populated with quality options.
Click/touch YouTube icon Browser should get referred to the video on YouTube.
