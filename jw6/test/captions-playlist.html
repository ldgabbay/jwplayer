Captions: Playlist test

List of test devices

This test implements the Captions feature in a playlist.

ActionExpected result(s)Works?
Play the first video The CC button should be visible in the controlbar.
Mouseover the CC button. A menu with (Off) and English should pop up.
Click the English option in the menu The video should display english captions at the right time.
Play the second video. The next video should start playing. Captions should not be displayed.
Mouseover the CC button in the controlbar The menu that pops up should contain 10 options. (Off) should be highlighted by default.
Select the Turkish option in the menu. The video should begin the display turkish captions. The CC menu should reflect this change.
