Documentation
=============

Documentation of this player, in RST format, can be found in the "doc" folder.
In "doc/_build" you'll find a rendered PDF of the docs.


Build from source
=================

To build the JW Player, you will need Adobe Flash CS4 or higher. 
The player.fla can be found in the "src" diractory.


Player testing
==============

The "test" directory contains a copy of our testing tool. 
At the top of index.html, you'll find a list with test settings.
In order to also test plugins / skins, make sure to download those too.
